# Vue.js Study

## 목표
Vue.js를 활용하여 Single Page App을 만들면서, Vue.js의 장단점을 파악해보고 최신 프론트엔드 개발 트렌드를 경험해본다.

## 학습 내용
1. npm, webpack, babel 사용법
2. javascript
   - es6~7 문법 학습
   - es5에서 달라진 점 위주로 학습
3. Vue.js
   - The vue instance
   - Template Syntax
   - Computed Properties and Watchers
   - Class and Style Bindings
   - Conditinal Rendering
   - List Rendering
   - Event Handling
   - Form input Bindings
   - Compoents
   - vue-class-component
   - Vue-Router
   - ajax (axios)
